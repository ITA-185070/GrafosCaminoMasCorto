# Grafos: Camino Más Corto

Interactive visualization of shortest path algorithms (**Dijkstra** and **Bellman-Ford**) using SageMath, Jupyter, and Voila.

### Local Execution

Ensure conda environment is active, then run:

```bash
voila CaminoMasCorto.ipynb --template=reveal
```

## Features

- **Step-by-step visualization**
